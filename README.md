# Escena-en-OpenGL
Project showing a complete 3D-scene made in OpenGL:

![2](https://user-images.githubusercontent.com/87211165/170189655-9ae2fee5-3d62-4008-8121-dab8a1f506e5.png)


It implements:

- Camera control
- Light source
- Shadows
- Blending
- Stencil
- Selection and controls
- NURBS
- Articulated objects
- Objects movement
