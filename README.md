# Escena-en-OpenGL
Proyecto en el que se muestra una escena 3D en opengl:

![2](https://user-images.githubusercontent.com/87211165/170189655-9ae2fee5-3d62-4008-8121-dab8a1f506e5.png)


Se implementa:

- Control de Cámara
- Fuentes de Luz
- Sombras
- Blending
- Stencil
- Selección y controles
- NURBS
- Objetos articulados
- Movimiento de objetos
